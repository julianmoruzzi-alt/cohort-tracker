⚙ Setup — Connect to Google Sheets

Follow these steps once. After setup, all staff will share live data stored securely in your school's Google Drive.

1

Create a new Google Sheet

Go to sheets.google.com and create a new blank spreadsheet. Name it something like Vulnerable Cohort Data. Keep the tab named Sheet1.

⚠️ Make sure you are signed in with your school Google account (not a personal Gmail), so the data stays within Gateway Academy's domain.
2

Open the Apps Script editor

In your new Google Sheet, click the menu: Extensions → Apps Script. A new tab will open with a code editor.

Delete everything in the editor, then copy and paste the entire script below:

const SHEET_NAME = 'Sheet1'; const ADMIN_PIN = '1234'; function doGet(e) { const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName(SHEET_NAME); const data = sheet.getDataRange().getValues(); if (data.length <= 1) return jsonResp([]); const headers = data[0]; const rows = data.slice(1).map(row => { const obj = {}; headers.forEach((h, i) => obj[h] = row[i]); try { obj.flags = JSON.parse(obj.flags || '[]'); } catch(e) { obj.flags = []; } try { obj.log = JSON.parse(obj.log || '[]'); } catch(e) { obj.log = []; } obj.att = Number(obj.att) || 0; obj.id = Number(obj.id) || 0; return obj; }); return jsonResp(rows); } function doPost(e) { try { const payload = JSON.parse(e.postData.contents); const action = payload.action; const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName(SHEET_NAME); if (action === 'save') { const pupils = payload.pupils; sheet.clearContents(); sheet.appendRow(['id','name','year','att','worker','flags','status','log']); pupils.forEach(p => { sheet.appendRow([ p.id, p.name, p.year, p.att, p.worker || '', JSON.stringify(p.flags || []), p.status || 'open', JSON.stringify(p.log || []) ]); }); return jsonResp({ok: true}); } if (action === 'delete') { if (payload.pin !== ADMIN_PIN) return jsonResp({ok: false, error: 'Wrong PIN'}); const id = Number(payload.id); const data = sheet.getDataRange().getValues(); for (let i = data.length - 1; i >= 1; i--) { if (Number(data[i][0]) === id) { sheet.deleteRow(i + 1); break; } } return jsonResp({ok: true}); } return jsonResp({ok: false, error: 'Unknown action'}); } catch(err) { return jsonResp({ok: false, error: err.toString()}); } } function jsonResp(data) { return ContentService.createTextOutput(JSON.stringify(data)) .setMimeType(ContentService.MimeType.JSON); }

After pasting, change ADMIN_PIN = '1234' on line 2 to a PIN only you know.

3

Deploy the script as a web app

In the Apps Script editor, click Deploy → New deployment.

In the dialog that opens:

• Click the gear icon next to "Type" and select Web app
• Set Execute as: Me
• Set Who has access: Anyone within [your school domain]
• Click Deploy

Google will ask you to authorise the script — click through and allow it (this is your own school account authorising your own script).

After deploying, you'll see a Web app URL. Copy it — it looks like:
https://script.google.com/macros/s/XXXXXX.../exec

4

Paste the URL and your PIN into this app

5

Share the HTML file and the Google Sheet with your staff

• Upload this HTML file to your school Google Drive and share it with your 2–5 staff (view or edit access on the file itself doesn't matter — they just need to be able to open it in Chrome).
• Share the Google Sheet with the same staff so they can see the underlying data if needed.
• All staff must be signed into their school Google account when using the app.

🔒 Data security reminder: This data is stored in your school's own Google Workspace for Education account and is subject to your school's data protection policies. Do not share the sheet or the app URL outside of authorised staff.
